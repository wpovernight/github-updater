# GitHubUpdater

This repository contains a modified version of the original GitHubUpdater created by Ryan Sechrest. The original project can be found at:

[https://github.com/ryansechrest/github-updater-demo](https://github.com/ryansechrest/github-updater-demo)

## About the Original Author

GitHubUpdater was initially developed by Ryan Sechrest to enable WordPress to check for and update custom plugins hosted on GitHub. It provides an efficient mechanism for integrating WordPress's plugin update system with GitHub-hosted repositories.

## Modifications by WP Overnight

This fork has been modified by WP Overnight to suit specific requirements and improve integration with our plugins.
